Quick Start

Get up and running in 3 simple steps

1

Install

2

Configure

# Create .env file GITLAB_TOKEN=your_gitlab_token LLM_API_KEY=your_llm_api_key LLM_PROVIDER=openrouter
3

Run

mr-pilot 1763 -i requirements.txt --comment
